# 🚀 快速开始指南

## 环境要求

- Python 3.9+
- 至少一个OpenAI兼容API的Key（推荐DeepSeek）

## 安装

```bash
git clone https://github.com/fangqian616/consensus-pipeline.git
cd consensus-pipeline
pip install -r requirements.txt
```

## 启动

```bash
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`。

## 第一次使用

### 方式一：智能配组（推荐新手）

1. 点击 **🧠 智能配组** 标签
2. 在文本框中描述你想制作的内容，例如：
   > "我想做一段沙漠中的机甲追逐战，有沙虫从地下冲出，主角在木板上射击"
3. 点击 **AI智能配组**
4. AI会自动分析你的需求，生成工作组配置
5. 你可以逐个编辑各部门的提示词，调整辩手
6. 点击 **确认并开始**，进入辩论流程

### 方式二：使用内置预设

1. 点击 **🧠 智能配组** 标签
2. 在预设下拉框中选择 **动画辩论**
3. 直接点击 **确认并开始**

### 方式三：导入已有配置

1. 点击 **🧠 智能配组** 标签
2. 点击 **📥 导入配置**
3. 粘贴之前导出的JSON配置
4. 点击 **确认并开始**

## 辩论流程

### Step 1: 输入剧本

在 **📝 输入** 标签中：
- 填写你的剧本/场景描述
- 添加正面提示词（想要的视觉风格）
- 添加负面提示词（不想要的元素）
- 可选：上传角色参考图

### Step 2: 部门辩论

在 **🗣️ 部门辩论** 标签中：
- 8个部门依次进行内部辩论
- 每个部门3-4位辩手对抗后达成共识
- 可以查看每位辩手的发言和最终共识
- 导演可以打回要求重辩

### Step 3: 交叉辩论

在 **⚔️ 交叉辩论** 标签中：
- 不同部门之间碰撞观点
- 消除跨部门的矛盾
- 形成统一的创作方向

### Step 4: 校对

在 **🔍 校对** 标签中：
- 编剧/空间/分镜/摄影/剪辑五部门审查最终产出
- 标注问题并自动修正

### Step 5: 获取产出

在 **🎬 最终产出** 标签中：
- **9宫格分镜表**：每格包含景别、角色位置、动作、叙事节拍
- **逐镜视频提示词**：复制即可用于AI视频生成工具
- **资产清单**：角色/道具/场景资产列表

## 保存你的配置

在智能配组页面修改完提示词后：
1. 点击 **💾 保存配置**
2. 输入名称（如"我的追逐战配置"）
3. 下次打开会自动加载

你也可以点击 **📋 导出配置** 将配置复制为JSON，分享给其他人。

## API配置

在侧边栏中：
- 选择模型（DeepSeek V4 Flash / Pro / 自定义）
- 填写API Key
- 自定义模式需填写API地址和模型名

## 常见问题

**Q: Token消耗大吗？**
A: 完整8部门辩论约消耗10k-30k token（取决于辩论轮次）。使用"专家池"模式可减少约50%。

**Q: 支持哪些API？**
A: 任何OpenAI兼容API都支持。推荐DeepSeek（性价比最高），也支持本地部署的模型。

**Q: 辩论结果质量不稳定怎么办？**
A: 1) 增加辩论轮次 2) 使用更强的模型 3) 在提示词中增加更具体的约束 4) 使用导演打回功能重辩

**Q: 可以只跑部分部门吗？**
A: 可以。在智能配组中禁用不需要的部门即可。
