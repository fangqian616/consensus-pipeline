## 🤔 这是什么？

你有没有想过——如果AI不是一个人埋头干活，而是像一个**真正的创意团队**一样，不同专业的人各抒己见、碰撞争论、达成共识？

Consensus Pipeline 就是这个想法的实现：

```
你的一句话需求
    ↓
AI自动组建工作组（编剧/分镜/摄影/灯光/VFX/音效/空间/剪辑）
    ↓
每个部门内部：多位辩手对抗 → 达成部门共识
    ↓
部门之间：交叉辩论 → 消除矛盾 → 统一方向
    ↓
最终产出：完整的分镜表 + 逐镜视频提示词
```

## ✨ 核心特性

### 🧠 智能配组
输入你的需求，AI自动分析内容类型、视觉风格、复杂度，为你组建最合适的工作组——哪些部门参与、每个部门派谁、用什么风格，全自动。

### ✏️ 全流程可编辑
AI配好的每个部门的提示词、辩手风格、参数都可以手动微调。改完直接跑，不满意的随时调。

### 💾 本地持久化
你的自定义配置自动保存到本地，下次打开直接用。内置预设 + 用户配置，两套体系互不干扰。

### 🎯 内置预设：动画辩论
开箱即用的8部门×24辩手完整动画制作管线，包含日漫视觉基因注入、空间定位系统、校对环节。从一段剧本到完整的9宫格分镜表+视频提示词，一键完成。

### 🔄 三种架构模式
| 模式 | 说明 | 适合场景 |
|------|------|---------|
| **共识管线** | 8部门串行辩论，全员参与 | 高质量产出，追求细节 |
| **专家池** | AI按场景精选2人/部门 | 快速产出，节省Token |
| **单Agent基线** | 单次LLM调用直接生成 | 对照实验，效率优先 |

### 🏪 市场模拟模型（Market Mode）
不是让辩手互相说服，而是让创意像商品一样在市场中竞争：

```
Step 1: 候选竞选 — AI为每个部门生成3个风格不同的候选方案
    ↓
Step 2: 质量叩问 — AI提出尖锐的质量评估问题
    ↓
Step 3: 投票选举 — 每个问题对3个方案进行投票，选出最佳
    ↓
Step 4: 补丁修正 — 对胜出方案的不足打补丁
```

**市场模式 vs 共识模式的核心区别：**

| | 共识模式 | 市场模式 |
|---|---------|---------|
| 辩手来源 | 预设固定辩手 | AI生成3个候选方案 |
| 竞争机制 | 辩论说服 | 投票选举 |
| 产出方式 | 多轮对抗→综合共识 | 候选竞选→质量叩问→胜出+补丁 |
| 适合场景 | 已有明确风格偏好 | 探索未知方向，需要多样性 |

市场模式特别适合"我不确定想要什么风格"的场景——让3个方案自己竞争，你只需要判断哪个更好。

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/fangqian616/consensus-pipeline.git
cd consensus-pipeline

# 安装依赖
pip install -r requirements.txt

# 启动
streamlit run app.py
```

打开浏览器后：
1. **Tab0 智能配组** — 描述你的需求，AI自动配组（或选择内置预设）
2. **Tab1 输入** — 填写剧本、正面/负面提示词、角色参考
3. **Tab2 部门辩论** — 8个部门依次辩论，每个部门3-4位辩手对抗后达成共识
4. **Tab3 交叉辩论** — 部门间碰撞，消除矛盾
5. **Tab4 校对** — 四部门审查最终产出
6. **Tab5 最终产出** — 9宫格分镜表 + 逐镜视频提示词，复制即用

## 🎮 辩论是怎么进行的？

以"动画辩论"预设为例：

### 部门辩论
每个部门有3-4位不同观点的辩手，比如编剧部：

| 辩手 | 立场 | 信条 |
|------|------|------|
| 微表情专精派 | 细节派 | 一个眼神比十句台词更有力 |
| 肢体语言专精派 | 动作派 | 身体语言暴露真实状态 |
| 情绪节奏专精派 | 节奏派 | 戏剧力量来自情绪转折的时机 |
| 叙事架构师 | 结构派 | 骨架搭好，血肉才有附着点 |

每位辩手基于自己的专业视角提出方案，多轮对抗后由"导演"（LLM）综合出部门共识。

### 交叉辩论
部门之间观点冲突怎么办？交叉辩论来解决：

```
编剧共识  ←→  分镜共识    （叙事节拍 vs 镜头切分）
摄影共识  ←→  灯光共识    （画面构图 vs 光影氛围）
VFX共识   ←→  音效共识    （视觉冲击 vs 听觉节奏）
空间共识  ←→  剪辑共识    （场景调度 vs 节奏控制）
```

### 最终产出
- **9宫格分镜表** — 每格包含：景别、角色位置、动作、叙事节拍、切换动机
- **逐镜视频提示词** — 每个镜头的完整AI视频生成提示词，支持中英文

## 🎨 创建你自己的预设

Consensus Pipeline 不只是动画制作工具——任何需要多维度创意碰撞的场景都适用：

- 🎬 **动画/短片制作** → 内置预设，开箱即用
- 📱 **产品宣传视频** → 市场部/创意部/摄影部/剪辑部
- 📝 **小说/剧本创作** → 人物派/情节派/节奏派/主题派
- 🎮 **游戏关卡设计** → 玩法派/叙事派/美术派/音效派
- 📊 **商业方案** → 数据派/创意派/风控派/执行派

通过Tab0智能配组，AI会根据你的需求自动选择部门和辩手风格。你也可以手动编辑每个辩手的提示词，保存为自己的专属预设。

## 📊 四种架构模式对比

```
共识管线（Pipeline）    → 高质量，多轮辩论，Token消耗较高
市场模拟（Market）      → 多样性，候选竞选+投票选举，探索性最强
专家池（Expert Pool）    → 快速，精选辩手，Token消耗适中  
单Agent基线（Baseline） → 最快，无辩论，Token消耗最低
```

| 指标 | 共识管线 | 市场模拟 | 专家池 | 单Agent |
|------|---------|---------|--------|---------|
| Token消耗 | ★★★★ | ★★★★★ | ★★ | ★ |
| 产出质量 | ★★★★★ | ★★★★ | ★★★ | ★★ |
| 风格多样性 | ★★ | ★★★★★ | ★★ | ★ |
| 速度 | ★★ | ★★ | ★★★ | ★★★★★ |
| 适合场景 | 追求极致质量 | 探索未知方向 | 快速迭代 | 对照基线 |

同一个输入分别跑不同模式，对比产出质量，量化"辩论"和"市场竞争"的价值。
