## 五、研究空白与机会

**1. 多任务联合建模** — 碳排放-电价-负荷缺乏联合优化框架 → 建议：多任务学习(MTL)
**2. 动态因果效应** — 因果ML在能源政策评估中起步 → 建议：DID+因果森林
**3. 跨领域迁移学习** — EU-ETS vs 中国碳市场模型迁移鲜有研究 → 建议：域自适应方法
**4. GNN网络效应** — 电力/碳市场图结构信息未利用 → 建议：图神经网络建模溢出效应
**5. 可解释性标配化** — 多数ML论文缺SHAP/LIME分析 → 建议：可解释性作为标准输出
