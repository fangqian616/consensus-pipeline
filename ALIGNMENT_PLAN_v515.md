# 共识管线报告质量对齐方案 v5.1.5

## 核心问题

当前报告读起来和"随便让AI写一篇综述"没有区别——泛泛而谈、没有数据、没有争议、没有深度。
共识管线的核心优势（多部门辩论+交叉质疑+反面证据）在报告中完全不可见。

## 根因分析

### 架构层（小夏负责）
1. **debate_outputs注入Bug** — v5.1.4辩论产出没传给综述AI，等于盲写 ✅已修复
2. **方法论匹配覆盖率92.5%"其他"** — 关键词太窄，大部分论文无法归类
3. **prompt不要求量化数据** — AI自然倾向于写"广泛应用""效果良好"这种废话
4. **续写截断** — 报告后半段被截，方法论对比表残缺
5. **饼图分类粗糙** — 能源经济学的方法论远不止LSTM/XGBoost这几类

### 内容层（科研助理负责）
1. **方法论分类体系不够** — 能源经济学特有的方法（分解-集成、混合模型、物理信息融合）没被识别
2. **量化对比标准缺失** — 没有告诉AI"每个方法必须引用具体精度/数据集/年份"
3. **辩论张力不可见** — 报告里看不到"文献检索组说A，反方质疑组说B"这种交锋
4. **研究空白缺乏文献计量支撑** — "没人做"只是主观判断，没有数据佐证

## 改动方案

### 一、报告结构重构

**旧结构**（像AI自动摘要）：
```
一、研究概况
二、方法论格局
三、核心发现
四、研究空白
五、参考文献
```

**新结构**（体现辩论过程）：
```
一、研究概况（数据卡片：检索N篇/S/A/B/时间跨度/方法类别数）
二、方法论演进与量化对比
   - 2.1 时间线：从统计→深度学习→混合→因果 的演进脉络（含年份节点图）
   - 2.2 量化对比矩阵：7列×7行（方法×评估维度），每个单元格必须有具体数字或文献引用
   - 2.3 辩论焦点：文献检索组vs反方质疑组对XXX方法的核心分歧
三、核心发现与争议（每个发现 = 支持证据链 + 反方质疑 + 交叉辩论结论）
四、研究空白与文献计量证据（每个空白 = 现状描述 + 为什么没人做(数据佐证) + 价值 + 可行路径）
五、参考文献（S/A/B分组）
```

### 二、Prompt强化（小夏改代码）

#### 2.1 综述prompt新增规则
```
规则16【量化硬规则】：描述任何方法效果时，必须包含至少一项量化指标
（如"预测精度MAPE=3.2%""在XX数据集上R²=0.95"），禁止只写"效果良好""广泛应用"。
若无具体数据，注明"该论文未报告量化指标"。

规则17【辩论焦点必现】：综述中至少3处引用辩论组的分歧点，
格式为"XX部门认为...，但反方质疑部门指出..."，
最终给出交叉辩论后的结论。

规则18【对比矩阵填充】：方法论对比表每个单元格必须非空，
预测精度列必须包含具体数值或"未报告"，
趋势列使用↑↑/↑/→/↓符号。

规则19【数据卡片先行】：综述开头必须先输出结构化数据卡片：
| 指标 | 值 |
| 检索论文数 | N |
| S级 | N |
| A级 | N |
| B级 | N |
| 时间跨度 | YYYY-YYYY |
| 方法类别数 | N |
| 预印本数 | N |
```

### 三、辩论产出展示格式（小夏改_format_debate_outputs）

当前：把辩手论点拼成一坨文本
改为：按"立场-论据-质疑-回应"结构化展示

```
【文献检索组】
├ 立场：深度学习在能源价格预测中精度领先
├ 论据：Shi(2017) Pooling RNN在家庭负荷预测RMSE降低15%[59]
├ 质疑(反方)：小样本下过拟合严重[53]
└ 回应：建议混合统计+深度学习方法[63]
```

### 四、内容层补充（科研助理负责）

1. **方法论7维度量化标准**：给出每个维度在能源经济学中的具体评判标准
   （如"预测精度"分档：MAPE<5%=优，5-10%=良，>10%=一般）
2. **反面证据强化prompt**：counter_evidence部门的prompt需要更具体的引导
3. **研究空白的文献计量佐证方法**：如何用已有论文数据证明"没人做"

### 五、报告质量验收标准

| 维度 | 及格线 | 良好 | 优秀 |
|------|--------|------|------|
| 量化数据覆盖 | >30%发现含数字 | >60% | >80% |
| 辩论焦点呈现 | ≥2处 | ≥3处 | 每个核心发现都有 |
| "参见[N]"残留 | ≤5处 | ≤3处 | 0处 |
| 方法分类覆盖 | "其他"≤40% | ≤25% | ≤15% |
| 反面证据 | ≥1条/发现 | ≥2条/发现 | 正反均衡 |
| 对比矩阵完整度 | ≥70%填充 | ≥90% | 100% |
| 报告完整性 | 5章全有 | 5章+数据卡片 | 5章+卡片+图注 |
